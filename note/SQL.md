# SQL

## 一、DQL(数据查询语句)

### 语法顺序

```sql
SELECT sum(col) FROM t WHERE col=v GROUP BY col HAVING col ORDER BY col limit begin_num count_num;
```

| 关键词   | 作用         |
| -------- | ------------ |
| SELECT   | 定义查询字段 |
| FROM     | 来自哪张表   |
| WHERE    | 筛选条件     |
| GROUP BY | 分组         |
| HAVING   | 过滤         |
| ORDER BY | 排序         |
| LIMIT    | 行分割       |

## 七、其他符号关键词用法

| 分类       | 符号\|关键词        | 作用                                                         |
| ---------- | ------------------- | ------------------------------------------------------------ |
| 通配符     | `*`                 | 字段通配符(代表所有)                                         |
|            | `_`                 | 字符通配符代表1个字符                                        |
|            | `%`                 | 字符通配符代表0个或多个字符                                  |
|            |                     |                                                              |
|            |                     |                                                              |
|            |                     |                                                              |
| 逻辑运算符 | `/`                 | 除法（分数符号）                                             |
|            | `*`                 | 乘法（在公式中）                                             |
|            | `+`                 | 加法（在公式中）                                             |
|            | `-`                 | 减法（在公式中）                                             |
|            | `=`                 | 等于                                                         |
|            | `>`                 | 大于, `>=`大于等于                                           |
|            | `<`                 | 小于, `<=`小于等于                                           |
|            | `<>`或`!=`          | 不等于                                                       |
|            | `between v1 and v2` | 在v1和v2之间的数，包括v1和v2                                 |
|            | `in`                | 条件范围筛选(列表)                                           |
|            | `not in`            | 不在该条件范围筛选                                           |
|            | `is null`           | 为空值                                                       |
|            | `is not null`       | 不为空值                                                     |
|            | `and`               | 与                                                           |
|            | `or`                | 或                                                           |
|            | `not`               | 非                                                           |
|            | `like`              | 模糊查询                                                     |
| 其他       | `as`                | 对表或字段起别名                                             |
|            | `distinct`          | 去除重复（放在`SELECT`后面）                                 |
|            | `()`                | 括号可以是列表`(v1, v2, v3)`，提供给in使用，也可以是调整运算的优先级`(col1+col2)` |
